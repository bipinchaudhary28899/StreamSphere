📺 Video Playback

  1. Angular requests feed from Node.js
  2. Node.js checks Redis (TTL 2 min)
  3. On miss → query MongoDB
  4. CloudFront URL returned in payload
  5. Browser streams video via CloudFront

⬆️ Video Upload

  1. Angular requests presigned S3 URL
  2. Node.js generates URL with UUID key
  3. Browser uploads directly to S3
  4. Angular calls /save-video
  5. Node.js calls HuggingFace for category
  6. Saved to MongoDB, Redis cache busted

🔍 Search & Filter

  1. User types → debounced 350ms
  2. Node.js checks Redis (TTL 60s)
  3. On miss → MongoDB regex query
  4. Category filter scopes results
  5. Cache keyed by term + category
